01 —

Speech-to-Text демо

Натиснете микрофона, говорете на български или английски, и наблюдавайте как речта ви се превръща в текст в реално време.

// STT Разпознаване на реч
Готов за запис
0 знака
Вашата реч ще се появи тук...
// Думи
0
// Изречения
0
// Увереност
—

// Използва Web Speech API (Chrome/Edge). Разпознаването е локално или чрез браузърен API — без изпращане към сървър.

02 —

Text-to-Speech демо

Въведете текст и изберете глас. Натиснете „Синтезирай" и чуйте синтезираната реч директно в браузъра.

// TTS Синтез на реч
0 знака
Зареждане на гласове...
1×
1×
Готов
Тук ще се показват думите при говоренето...

// Използва Web Speech Synthesis API. Наличните гласове зависят от операционната система и браузъра.

03 —

Пълен цикъл: Реч → Текст → Реч

Говорете — речта ви ще бъде разпозната (STT), след което автоматично ще бъде синтезирана и изречена обратно (TTS). Целият STT → TTS конвейер в едно.

// FULL PIPELINE STT → Текст → TTS
Натиснете микрофона за начало
1. Запис
→
2. STT
→
3. Текст
→
4. TTS
→
5. Говор
Речта ви ще се появи тук...
Текстът ще бъде изречен автоматично...
Историята ще се появи тук...

// При спиране на записа (или пауза в речта), разпознатият текст автоматично се предава към TTS.

04 —

Бележки за демото

// STT — Как работи тук

1

Web Speech API: Браузърът изпраща аудио потока към разпознавателен двигател (Google за Chrome, Microsoft за Edge).

2

Реално време: Резултатите идват постепенно — interim (временни) и final (окончателни) транскрипции.

3

Поддръжка: Работи в Chrome, Edge и Safari. Firefox не поддържа Web Speech API.

// TTS — Как работи тук

1

Speech Synthesis API: Браузърът използва системни гласове (инсталирани на ОС) за синтезиране на реч.

2

Управление: Контролирате скоростта, тона и избора на глас. Различни ОС имат различни налични гласове.

3

Highlighting: Демото маркира думите в реално време докато се изговарят.
